برمجيات وأدوات محاكاة

الرئيسية / برمجيات وأدوات محاكاة

أدوات رقمية مفيدة

مجموعة من البرامج والأدوات التي قد تكون مفيدة للطلاب والمعلمين في دراسة الفيزياء وإعداد المحتوى التعليمي.

قارئات ومستعرضات أساسية

  • Adobe Acrobat Reader DC

    قارئ ملفات PDF المجاني والأكثر شيوعًا، ضروري لعرض العديد من الوثائق والمذكرات.

    Windows macOS Mobile مجاني

أدوات إنتاجية

  • Microsoft 365 (Office)

    حزمة مكتبية تتضمن Word, Excel, PowerPoint، مفيدة لإعداد التقارير، العروض التقديمية، وتحليل البيانات.

    Windows macOS Web مدفوع (اشتراك)
  • 7Zip

    برنامج مجاني ومفتوح المصدر لضغط وأرشفة الملفات بنسبة ضغط عالية (خاصة لصيغة 7z)، ويدعم فك ضغط العديد من الصيغ الأخرى.

    Windows Linux (p7zip) مجاني ومفتوح المصدر

برامج علمية متخصصة

  • Scidot Math Physique

    إضافة لبرنامج Microsoft Word تسهل كتابة المعادلات الرياضية والرموز الفيزيائية والكيميائية بشكل احترافي.

    Windows (Word Add-in) مفعل
  • GeoGebra

    برنامج مجاني للرياضيات والعلوم يتيح إنشاء رسوم بيانية، محاكاة هندسية، وجداول بيانات، مفيد في تصور بعض المفاهيم الفيزيائية.

    Win Mac Linux Web Mobile مجاني
  • MathType

    محرر معادلات رسومي قوي لإنشاء الرموز الرياضية المعقدة لإدراجها في المستندات وصفحات الويب وتطبيقات أخرى.

    Windows Windows (Word Add-in) مفعل

أدوات محاكاة تفاعلية

  • PhET Interactive Simulations

    مجموعة ضخمة من المحاكاة التفاعلية المجانية في الفيزياء والكيمياء والرياضيات وعلوم الأرض والأحياء (جامعة كولورادو).

    Web (HTML5) مجاني
