LibreOffice

هي حزمة مكتبية قوية، مجانية، ومفتوحة المصدر، تم تطويرها بواسطة The Document Foundation. تعتبر بديلاً ممتازاً وشاملاً لحزمة Microsoft Office، وتوفر مجموعة كاملة من التطبيقات المكتبية لإنشاء وتحرير المستندات النصية، جداول البيانات، العروض التقديمية، الرسومات، قواعد البيانات، والمعادلات الرياضية.

المكونات الرئيسية:

  • **Writer:** معالج نصوص مشابه لـ Microsoft Word.
  • **Calc:** برنامج جداول بيانات مشابه لـ Microsoft Excel.
  • **Impress:** برنامج عروض تقديمية مشابه لـ Microsoft PowerPoint.
  • **Draw:** برنامج رسومات متجهية ومخططات انسيابية.
  • **Base:** برنامج لإدارة قواعد البيانات.
  • **Math:** محرر معادلات وصيغ رياضية.

ميزات إضافية:

  • توافق عالي مع صيغ ملفات Microsoft Office (.docx, .xlsx, .pptx) وغيرها الكثير.
  • دعم أصلي لتنسيق OpenDocument Format (ODF) القياسي المفتوح.
  • واجهة مستخدم قابلة للتخصيص (بما في ذلك واجهة تشبه الشريط).
  • مجتمع تطوير ودعم نشط.
  • توفر قوالب وإضافات لتوسيع الوظائف.

الأنظمة المدعومة:

  • Windows
  • macOS
  • Linux

راجع الموقع الرسمي لمتطلبات النظام المحددة لكل إصدار.
